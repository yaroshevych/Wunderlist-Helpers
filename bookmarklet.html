Install Wunder Print boormarklet

A bookmarklet is JavaScript stored as the URL of a bookmark in a web browser. Wunder Print bookmarklet will fetch all tasks from all your lists and display them on single page. You'll be able to print or save that list for backup purposes.
Wunder Print
